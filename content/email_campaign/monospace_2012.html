If you think that Monospace is only about Mono, you’re mistaken. We’re working on changing that perception!

Monospace is the official cross platform and open-source .NET conference. Want to learn more about developing for the iPhone, Android, Mac, and *nix platforms using .NET technologies? How about developing games or learning more about open-source projects using .NET technologies? Monospace has provided an annual venue to collaborate, share, and socialize around these topics and more. Monospace is more than just about Mono, it's geared toward the whole .NET open-source ecosystem. We have exciting news to share about that at the conference as well!
